# Google Atelierul Digital project

## Simple web site created with django framework for managing a record label.


## The web site has 2 web scrapers (<a href="https://user-images.githubusercontent.com/61471997/132495313-d2cde281-41d5-4995-acab-4675fbf54244.jpg">news</a> + <a href="https://user-images.githubusercontent.com/61471997/132495374-4cec7300-2ffe-4c50-8959-e8ec51485cf4.jpg">billboard</a>).

## CRUD for <a href="https://user-images.githubusercontent.com/61471997/132495755-586bf33d-27a4-421e-8677-e4e7f8b02987.jpg">artists</a>, <a href="https://user-images.githubusercontent.com/61471997/132495764-03c6fa91-12ca-4bf8-b908-a2a8ad12fa9f.jpg">songs</a>, albums, collaborations, locations, concerts, contracts.

## And a simple admin <a href="https://user-images.githubusercontent.com/61471997/132495912-8f815ad3-92f5-4987-b76a-ac3e2eaa18e1.jpg">page</a>.

Credentials for admin page:

## To run the app you have to activate the <a href="https://user-images.githubusercontent.com/61471997/132496372-64a2d9cb-321d-4406-ac34-f84aa2cc900e.png">virtual enviroment</a> and run the app.

In the record_label_project directory run the followings commands.
```
 ..\venv\Scripts\activate
 py .\manage.py runserver
```
